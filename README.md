# dotfiles

## vim配置文件

### 插件
使用Vundle管理插件，目前使用的插件有

 - YCM,代码补全插件
 - YCM-Generator　用于YCM配置文件生成
 - nerdtree 文件导航插件
 - vim-clang-format 代码格式化插件
 
### 快捷键

`ctrl+n`  打开\关闭　nerdtree导航

`ctrl+@+y` 生成ycm配置文件，需要在项目根目录下运行

`ctrl+@+g`　跳转到定义　           （cscope支持)

`ctrl+@+c`  跳转到调用此函数的地方 （cscope支持)

`ctrl+@+s`  根据字符串跳转         （cscope支持)
